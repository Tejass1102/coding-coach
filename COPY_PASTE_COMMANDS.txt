COPY-PASTE COMMANDS FOR CODING COACH SETUP
==========================================

STEP 1: ENVIRONMENT SETUP
=========================

cd D:\College\DL\coding-coach
& .\venv\Scripts\Activate.ps1
pip install -r backend\requirements.txt
pip install sentence-transformers scikit-learn


STEP 2: BUILD DATASET  
=====================

*** IMPORTANT: Replace these with YOUR actual paths ***
- YOUR_USERNAME: Your Windows username
- REPO_PATH: Full path to where you cloned leetcode-java

cd D:\College\DL\coding-coach\backend\notebooks

python build_dataset.py `
  --leetcode-csv "C:\Users\YOUR_USERNAME\Downloads\Leetcode_Questions_updated (2024-11-02).csv" `
  --leetcode-java-clone "D:\REPO_PATH\leetcode-java" `
  --output-dir ..\datasets `
  --spot-check-per-label 3


STEP 3: TRAIN MODEL
===================

python train_classifier.py `
  --dataset-jsonl ..\datasets\train.jsonl `
  --epochs 50 `
  --batch-size 8 `
  --learning-rate 0.0005


STEP 4: VERIFY SETUP
====================

cd D:\College\DL\coding-coach
python verify_setup.py


STEP 5: START BACKEND (keep this terminal open)
================================================

python backend\main.py


STEP 6: TEST API (in a NEW terminal)
====================================

python test_api.py


OPTIONAL: AUTOMATED SETUP (instead of steps 1-3)
=================================================

** Update these paths in auto_setup.ps1 first **

powershell -ExecutionPolicy Bypass -File auto_setup.ps1

COMMON ISSUES & SOLUTIONS
==========================

Issue: "Module not found"
Solution: pip install sentence-transformers scikit-learn

Issue: "CSV file not found"
Solution: Check Windows username matches YOUR actual username in path

Issue: "Cannot connect to backend"
Solution: Start backend: python backend\main.py

Issue: "Model not found"
Solution: Run training: cd backend\notebooks && python train_classifier.py --dataset-jsonl ..\datasets\train.jsonl

Issue: "Dataset not found"
Solution: Build dataset: cd backend\notebooks && python build_dataset.py --leetcode-csv "..." --leetcode-java-clone "..." --output-dir ..\datasets

WHAT EACH COMMAND DOES
======================

1. activate.ps1           - Activates Python virtual environment
2. pip install           - Installs Python packages
3. build_dataset.py      - Creates training data from LeetCode
4. train_classifier.py   - Trains the ML model
5. verify_setup.py       - Checks if everything is installed
6. main.py               - Starts the backend server
7. test_api.py           - Tests API endpoints
8. auto_setup.ps1        - Runs all steps automatically
